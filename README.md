# EVM contracts for Bridge (bridge.pepe.team)

## Usage

### Pre Requisites

Before running any command, make sure to install dependencies:

```sh
npm i
```

### Compile

Compile the smart contracts with Hardhat:

```
npm run compile
```

## Tests

Run the tests:

```
npm run test
```

## Deploy local

```
npx hardhat node --tags "local" --network hardhat
```
